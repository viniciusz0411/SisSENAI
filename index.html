SisSenai 1.0-Unidade Capanema

Sistema desenvolvido para integrar aprendizado técnico e gestão comercial, conectando Web e Desktop ao banco SQLITE.

🚀Módulos do Sistema

Ícone Descrição
👫 Clientes: Cadrasto e histórico de atendimentos.
📦 Vendas: Controle de produtos e reposição.
🏅 Vendas: Registro de transações e orçamentos.

📊Tecnologias

  • Backend: Node.js com Express.
  • Banco: SQLite (sissenai.db).
  • Desktop: Visual Basic .NET (Módulo de conexão).
